# Aplikacja wizualizująca ukształtowanie terenu (LIDAR) / Terrain visualization application (LIDAR)

(PL) Aplikacja służaca do wizualizacji 3D danych z plików TIF. Wykorzystuje ona bilbioteki Three.js i geotiff.js do prezentacji ukształtowania terenów w Wielkiej Brytanii. Folder 'tiles' domyślnie powinien zawierać obrazy TIF z odpowiednimi nazwami (TR26se_DTM_1m.tif i podobnym). Aplikacja ma zaimplementowane prezentację wysokości w postaci poziomych linii co 50 metrów oraz panel do przechodzenia między kolejnymi wycinkami TIF.

(EN) An application for 3D visualization of data from TIF files. It uses the Three.js and geotiff.js libraries to present the terrain of the UK. The 'tiles' folder should contain TIF images with appropriate names (TR26se_DTM_1m.tif or similar) by default. The application has implemented elevation display in the form of horizontal lines every 50 meters and a panel for navigating between TIF slices.
